Daily Flow

The height of a single bar is the total number of people suffered from Coronavirus in Hawaii and confirmed to be infected. It includes three parts: those who could or could not recover and those who are currently in the active phase of the disease.

New Confirmed Cases

This graph shows the number of new cases by day. The lightblue bars are the number of the new total confirmed cases appeared that day.

Daily Speed

This graph shows the speed of growth (in %) over time in Hawaii. The only parameter here is the number of confirmed cases.

Note. When the speed is positive, the number of cases grows every day. The line going down means that the speed decreases, and while there may be more cases the next day, the disease spread is slowing down. If the speed goes below zero, that means that fewer cases registered today than yesterday.

Cumulative 14-day index

This graph shows the index that some countries use to estimate how safe travelling is. The value is the number of new confirmed cases for the last 14 days per 100,000 of population.

123,251 tests performed

This graph shows the total number of tests performed in Hawaii. This graph does not reflect the outcome of the test cases. 9.06 % of the whole population have been tested.

Weekly Levels

This graph draws the number of deaths in Hawaii connected to the COVID-19 infection aggregated by weeks of 2020.

Per capita values

Here, the number of confirmations and deaths per 1000 of population in Hawaii is shown. These numbers is a better choice when comparing different countries than absolute numbers.
